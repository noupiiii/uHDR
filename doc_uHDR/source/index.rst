Documentation uHDRv6
===================

uHDRv6 est un logiciel d'édition d'images HDR (High Dynamic Range) développé par Rémi Cozot. 
Cette version dispose d'un cœur de traitement écrit en C++ et d'une interface graphique développée en Python avec PyQt5.

Le logiciel est conçu pour traiter et améliorer des images à haute gamme dynamique, permettant de manipuler 
des images avec un contraste, une luminosité et une richesse de couleurs bien supérieurs aux images standard.
